# 嘘嘘 AI 日报 🤫

西西的电子秘书「嘘嘘」每天自动生成的 AI 领域精选日报。

## 功能

- 📡 **自动抓取**：每天北京时间 8:00 自动运行
- 📰 **6 大新闻源**：36kr、量子位、机器之心、Hacker News、MIT Technology Review、VentureBeat AI
- ⭐ **GitHub Trending**：抓取近 7 天 AI/LLM 相关高星项目
- 🤖 **智谱 GLM-4.7-Flash 总结**：精选 5-8 条，按分类整理
- 🙋 **小白解读**：每条附有非技术背景友好的白话解释
- 💡 **relevance 评分**：说明每条资讯和西西工作/学习的关系

## 日报分类

- 🤖 大模型
- 🦾 具身智能
- 🛠️ 工具
- 📄 论文/研究
